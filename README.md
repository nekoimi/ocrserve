```shell
# Test
pip install -r requirements-test.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/

```

## Install it from PyPI

```bash
pip install ocrserve
```

## Usage

```py
from ocrserve import BaseClass
from ocrserve import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m ocrserve
#or
$ ocrserve
```
